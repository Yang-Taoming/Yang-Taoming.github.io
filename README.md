# Yang Taoming Portfolio

Personal portfolio website for an AI algorithm engineer profile, built with React and Vite.

## Local Preview

```bash
cd /Applications/杨焘溟/Yang-Taoming.github.io
npm_config_strict_ssl=false npm install --no-audit --no-fund --cache ./.npm-cache
npm run dev -- --port 5173
```

Open:

```text
http://localhost:5173/
```

## Production Build

```bash
npm run build
```

The production files will be generated in `dist/`.
